# Errands

A personal, $0-forever iPhone app: add errands by voice, get speed-aware reminders
near any matching store branch anywhere, with all data staying on-device. Built as a
native Swift/SwiftUI app, compiled unsigned on GitHub Actions (there is no Mac in this
project), and sideloaded via AltStore with a free Apple ID.

See [DESIGN.md](DESIGN.md) for the full design and [docs/plans/](docs/plans/) for the
implementation plan.
